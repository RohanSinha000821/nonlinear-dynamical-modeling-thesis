# Nonlinear Dynamical Modeling of Brainwave Synchronization and Learning Dynamics

Integrated M.Sc. Thesis  
Birla Institute of Technology, Mesra (2025)

## Overview
This thesis develops a three-dimensional nonlinear dynamical system modeling the interaction between neural synchronization, cognitive load, and adaptive learning performance.
